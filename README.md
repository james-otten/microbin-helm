# microbin-helm

Unofficial helm chart for the [microbin](https://github.com/szabodanika/microbin) application.

## Downloading the Chart
The charts are hosted at https://microbin-helm.jamesotten.com. You can add the Helm repo with:
```
helm repo add microbin https://microbin-helm.jamesotten.com
```
